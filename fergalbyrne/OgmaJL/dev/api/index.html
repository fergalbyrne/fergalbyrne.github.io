OgmaJL.locateWeight — Method
locateWeight(m::FocalMatrix, index::XW)
locateWeight(m::FocalMatrix, index::Int) = locateWeight(m, XW(index))

Using m.osize dimensions, finds the input position [ix iy iz] and the output position [ox oy oz] for the given weight index.

Examples

m = FocalMatrix(XIO3(8,10,3), XIO3(5,4,3), 3);

julia> i, o = locateWeight(m, 4142)
(XIO3[4×5×2], XIO3[4×3×1])

OgmaJL.sparse — Method
sparse(m::FocalMatrix, outs::Vector{AW})

Using m.osize dimensions, chooses the top entry in the z-column for each x-y position in outs, returning a one-hot tensor (zero elsewhere) and an array of winner z-indices for each x-y position.

Examples

import Random # hide
Random.seed!(1234) # hide
m = FocalMatrix(XIO3(3, 4, 3), XIO3(4, 5, 3), 1);
Random.seed!(56789) # hide
ins = AW.(rand([-1,1], 3 * 4 * 3) .* rand(3 * 4 * 3))
outs = zeros(AW, 4 * 5 * 3)
outs = forward!(f, ins, outs)
julia> sparseouts, tops = sparse(f, outs)
