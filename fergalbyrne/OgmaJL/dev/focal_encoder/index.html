The SparseFocalEncoder locally focussed encoder type

The SparseFocalEncoder connects a number of input tensors to a single hidden activation tensor, using an appropriate number of FocalMatrix weight matrices.

abstract type AbstractEncoder end

struct SparseFocalEncoder <: AbstractEncoder
    inputs::Vector{Vector{AW}}
    outputs::Vector{AW}
    weight_tensors::Vector{FocalMatrix}
end

There are three fields for the type.

inputs

A vector of input tensors.

outputs

The output activation tensor.

weight_tensors

A Vector of FocalMatrix, one for each input tensor.
