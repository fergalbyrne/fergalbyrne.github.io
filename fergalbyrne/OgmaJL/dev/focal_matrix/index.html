The FocalMatrix locally focussed weight matrix type

The FocalMatrix is a weight matrix which connects an input tensor ins of size isize (isize.x×isize.y×isize.z) to an output tensor outs of size osize (osize.x×osize.y×osize.z), where each output receives input from units in an x-y window within radius units of the input unit "below" the x-y of the output unit.

struct FocalMatrix
    isize::XIO3
    osize::XIO3
    radius::XIO
    xs::Array{XIO, 2}
    ys::Array{XIO, 2}
    idxs::Array{XW, 2}
    ws::Array{AW, 1}
    nWs::XW
end

There are nine fields for the type.

isize

The isize field specifies the dimensions of the layer's inputs (or visible layer) in x, y and z.

osize

The osize field specifies the dimensions of the layer's outputs (or hidden layer) in x, y and z.

radius

The radius field specifies the radius of a square x-y window in the inputs which are connected to any unit in the output layer.

xs (derived)

The xs field contains the minimum and maximum x edges of the input window for each x-y of the outputs. This field is computed by the external constructor below.

ys (derived)

The ys field contains the minimum and maximum y edges of the input window for each x-y of the outputs. This field is computed by the external constructor below.

idxs (derived)

The idxs field contains the starting weight index for a given output x-y-0 position. This field is computed by the external constructor below.

ws (derived)

The ws field contains the randomly initialised weight matrix. This field is computed by the external constructor below.

nWs (derived)

The nWs field is the size of the weight matrix. This field is computed by the external constructor below.
